**TITLE: Matrix Arithmetic Calculator**



**Author: Alexx William**



1. This program can perform simple arithmetic operations by taking two 3X3 order matrices A and B as an input from the user.
2. The programs can take up to 9 integers.
3. User can select the mode of operations such as - addition, subtraction and multiplication.
4. The program then presents the output in 3X3 order matrix form.
